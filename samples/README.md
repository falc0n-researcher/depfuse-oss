# Samples

HTML report samples generated by Depfuse.

| File | Command used |
|------|-------------|
| [scan.html](scan.html) | `depfuse scan <project> --format html` |
| [package.html](package.html) | `depfuse package next@15.1.0 --format html` |
| [cve.html](cve.html) | `depfuse cve CVE-2025-29927 --format html` |

These are real outputs generated from the demo package fixture included in this repository.
