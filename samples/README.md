# Samples

HTML report samples generated by Depfuse.

| File | Command used |
|------|-------------|
| [scan.html](scan.html) | `make samples` → `depfuse scan demo_package/` |
| [package.html](package.html) | `make samples` → `depfuse package next@15.1.0` |
| [cve.html](cve.html) | `make samples` → `depfuse cve CVE-2025-29927` |

Regenerate with `make samples` (requires seeded `testdata/intel.db` via `make testdata`, which fetches npm metadata for fixture and lookup trees — needs network).
